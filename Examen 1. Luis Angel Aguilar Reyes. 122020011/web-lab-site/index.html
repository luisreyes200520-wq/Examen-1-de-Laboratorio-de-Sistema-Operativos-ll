Sobre esta Práctica

Este sitio web fue publicado utilizando un contenedor Nginx ejecutado en Docker, montado mediante un volumen que enlaza esta carpeta local con la ruta de publicación del servidor (/usr/share/nginx/html). Esto permite modificar el contenido del sitio sin necesidad de reconstruir ni modificar directamente el contenedor.
